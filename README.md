## Notes app for daily notes, and daily quote for daily motivation :)

The application implements the functions:
- note Creation
- note Reading
- note Update 
- note Deletion
- note pin
- selecting the color and theme of a note
- daily quote

## Technologies and Tools

- ### UIKit
- ### Realm 
- ### SnapKit
- ### Swift Package Manager
